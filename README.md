# Probably Important

Probably Important is a note-taking app built with Next.js, TypeScript, Tailwind CSS, Bun, Prisma, Neon PostgreSQL, better-auth, and TipTap.

## Getting Started

1. Install dependencies.

   ```bash
   bun install
   ```

2. Start the development server.

   ```bash
   bun dev
   ```

3. Open the app in your browser.

   ```text
   http://localhost:3000
   ```

## Available Scripts

- `bun dev` starts the local development server.
- `bun run build` creates a production build.
- `bun run start` runs the production server.
- `bun run lint` runs ESLint.

## Project Goal

The app is intended to let authenticated users create, edit, delete, search, and share notes. Each note should include a title, content, and timestamp, and the editor should support rich text formatting through TipTap.
